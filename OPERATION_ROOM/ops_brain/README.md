# Ops Brain

Outcome-oriented operations brain for a living SOP system.

## Principle

Organize SOPs by business outcomes (for example: sales cycle and fulfillment), not by org roles.

## Outcomes

- `sales_cycle`
- `fulfillment`
- `ops_handoff`

## Execution

Use the command router:

- `/home/t79/KITTY/bin/ops-brain.sh list`
- `/home/t79/KITTY/bin/ops-brain.sh run sales_cycle`
- `/home/t79/KITTY/bin/ops-brain.sh run fulfillment`
- `/home/t79/KITTY/bin/ops-brain.sh run ops_handoff`
