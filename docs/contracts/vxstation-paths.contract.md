# VXSTATION Canonical Path Contract

## Canonical Root

`/local/VXSTATION`

## Canonical Subpaths

- `/local/VXSTATION/ASUSX`
- `/local/VXSTATION/VYRDX`
- `/local/VXSTATION/VYRDOX`
- `/local/VXSTATION/EVIDENCE`
- `/local/VXSTATION/ARCHIVE_ROOM`

## Boundary Rule

VXSTATION is the root workspace and local operating root.  
VXSTATION is not the live production runtime authority.

## Activation Note

If `/local` is not writable for the current user, path creation requires root privileges.
