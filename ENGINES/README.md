# ENGINES

Internal-only engine lane. Not publicly exposed.

Primary authority is code under:

- `/opt/kitty/ENGINES/index.ts`
- `/opt/kitty/ENGINES/boot.ts`
- `/opt/kitty/ENGINES/types.ts`
- `/opt/kitty/ENGINES/base.ts`
